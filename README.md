## How to Build and Run

### Prerequisites
- CMake (version 3.10 or higher)
- C++ compiler with C++17 support (GCC, Clang, MSVC)

### Building the Game
```bash
# Create build directory
mkdir -p build

# Generate build files
cd build
cmake ..

# Build the game
cmake --build .

# Run the game
./bin/dungeon_crawler
```
